# Bilillee

Bilillee is a booking marketplace for barbershops and salons with four core roles:

- Super Admin
- Shop Owner
- Staff
- Customer

## Current Direction

The app is being refined in phases so we can build it together without losing scope.

- Phase 1: MVP foundation
- Phase 2: operational depth
- Phase 3: production readiness

See [PHASES.md](./PHASES.md) for the working checklist.

## Stack

- React
- TypeScript
- Vite
- Supabase

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
